# Ford Car Price Predictor

## Overview
This project predicts the price of a Ford car using a Machine Learning model built with Linear Regression.

## Technologies Used
- Python
- Streamlit
- Pandas
- Scikit-learn
- Joblib

## Files
- app.py
- LR_ford_car.pkl
- scaler.pkl
- columns.pkl
- requirements.txt
- ford.csv

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
